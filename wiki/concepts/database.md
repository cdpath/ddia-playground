# Database

**Database** 是让应用（或其他应用）能够存储数据并在之后重新找到它的系统。它是 [[data-intensive-application]] 的基础组件之一。

不同 database 针对不同访问模式优化，包括 [[online-transaction-processing|OLTP]]、[[online-analytical-processing|OLAP]] 和嵌入式场景。它们支持不同的 [[data-model]]，如 [[relational-model]]、[[document-model]]、[[graph-data-model]] 等。本书后续章节会讨论 storage engine、replication、partitioning 和 transaction。
