# Semantic Web

**Semantic Web** 是 2000 年代初提出的愿景：让互联网上的数据不仅对人类可读，还以标准化、机器可读的格式发布，便于跨站点数据交换。

## 技术遗产

虽然原始 Semantic Web 愿景未完全成功，但其思想和技术留下了深远影响：

- Linked data 标准，如 JSON-LD。
- 生物医学本体（ontologies）。
- Facebook 的 Open Graph protocol。
- 知识图谱，如 [[wikidata]]。
- Schema.org 的结构化词汇。
- [[triple-store]] 和 [[sparql]] 在企业内部应用。
