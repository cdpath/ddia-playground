# JSONPath

**JSONPath** 是用于在 JSON 文档中定位数据的查询表达式，类似于 XML 的 XPath。

## 用途

- 从嵌套 JSON 中提取字段。
- 在 document database 或 API 响应中做轻量级查询。

## 与 XPath 的关系

JSONPath 借鉴了 XPath 的路径语法，但针对 JSON 的树形结构做了调整。
