# Service-oriented architecture (SOA)

**Service-oriented architecture（SOA）** 是将应用构建为一组通过网络通信的服务的较早期术语。[[microservices]] 是这一思想的近期精炼版本。

核心思想相同 — 把系统分解为带有明确 API 的网络服务 — 但 SOA 与 microservices 在规模、组织语境和工具链上有所不同。
