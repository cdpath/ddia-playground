# Edgar F. Codd

**Edgar F. Codd**（1923–2003）是计算机科学家，关系型数据库理论奠基人。

## 主要贡献

- 1970 年发表论文 *A Relational Model of Data for Large Shared Data Banks*，提出 [[relational-model]]。
- 奠定了现代 SQL 和 RDBMS 的理论基础。
