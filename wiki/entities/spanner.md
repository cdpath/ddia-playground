# Spanner

**Spanner** 是 Google 的全球分布式 relational database，支持同步复制、外部一致性事务和 SQL 查询。

## Chapter 3 角色

- Spanner 允许在 relational schema 中声明子表行嵌套在父表行中，从而提供类似 document model 的 data locality。
