# Apache DataFusion

**Apache DataFusion** 是用 Rust 实现的开源可嵌入 query engine。它提供 SQL 和 DataFrame API，支持 [[column-oriented-storage|列式]]、[[vectorized-processing|向量化]] 执行。

## 本章角色

- 新兴的开源 query engine 组件。
- 可作为独立工具或嵌入其他系统使用。
- 常与 [[apache-arrow]] 和 [[parquet]] 配合使用。
