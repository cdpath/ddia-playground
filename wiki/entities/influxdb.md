# InfluxDB

**InfluxDB** 是流行的时间序列数据库。其新版 IOx 存储引擎基于 Apache Arrow 和 [[column-oriented-storage|列式存储]]。

## 本章角色

- 列式存储应用于时序数据库的例子。
- IOx 引擎使用 Apache Arrow 作为内存格式，Parquet 作为持久化格式。
