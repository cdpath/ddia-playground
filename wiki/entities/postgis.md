# PostGIS

**PostGIS** 是 [[postgresql]] 的空间数据库扩展，为 Postgres 添加地理空间对象、空间函数和空间索引支持。

## 本章角色

- 使用 PostgreSQL 的 GiST 索引实现 [[r-tree]] 风格的空间索引。
- 是关系型数据库中处理地理空间数据的常用方案。
