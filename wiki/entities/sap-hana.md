# SAP HANA

**SAP HANA**（High-Performance Analytic Appliance）是 SAP 开发的 [[in-memory-database]]，同时支持事务处理和分析查询，属于 [[hybrid-transactional-analytical-processing|HTAP]] 系统。

## 本章角色

- 与 Microsoft SQL Server、SingleStore 并列为 HTAP 数据库。
- 内存优先架构，用于 SAP 生态中的实时分析和事务处理。
