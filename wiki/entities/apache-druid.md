# Apache Druid

**Apache Druid** 是面向实时分析的开源分布式数据库，广泛用于事件驱动数据的 OLAP 查询。

## 本章角色

- 与 Pinot、ClickHouse 并列为实时分析系统。
- 采用 [[column-oriented-storage|列式存储]]。
- 支持高吞吐实时摄取和低延迟聚合查询。
