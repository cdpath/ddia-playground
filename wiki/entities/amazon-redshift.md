# Amazon Redshift

**Amazon Redshift** 是 AWS 提供的 cloud [[data-warehouse]]，基于 PostgreSQL 衍生而来。它针对大规模分析查询优化。

## 本章角色

- 云原生 data warehouse 的代表之一。
- 采用 [[column-oriented-storage|列式存储]]。
- 支持从 S3 等对象存储直接查询（Redshift Spectrum）。
