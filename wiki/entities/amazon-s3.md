# Amazon S3

Amazon Simple Storage Service（S3）是 AWS 提供的 [[object-storage]] 服务，用简单 API 存储大文件，隐藏底层物理机器并自动分布数据。

## 本章角色

- 高级 cloud service 所依赖的低层服务例子
- Snowflake 被指出依赖 S3 存储数据
- 代表从 filesystem 向 managed object storage 的转变，是 [[cloud-native]] 架构的基础
