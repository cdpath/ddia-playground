# Unity Catalog

**Databricks Unity Catalog** 是 Databricks 平台提供的统一 data catalog 和治理解决方案，用于管理数据湖中的表、权限和血缘关系。

## 本章角色

- 现代解耦式 [[data-catalog]] 的例子。
- 与 Snowflake Polaris 并列为代表性 catalog 服务。
