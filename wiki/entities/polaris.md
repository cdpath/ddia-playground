# Polaris

**Snowflake Polaris** 是 Snowflake 提供的 data catalog 服务，用于管理和发现数据湖中的表和元数据。

## 本章角色

- 现代解耦式 [[data-catalog]] 的例子。
- 通常以独立服务形式运行，可通过 REST 接口访问。
