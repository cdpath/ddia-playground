# Apache Hive

**Apache Hive** 是 Hadoop 生态中的数据仓库软件，提供类 SQL 接口（HiveQL）查询 HDFS 上的大规模数据。

## 本章角色

- 早期把 query engine、storage format、table format 和 catalog 集成在一起的系统。
- 随着 data lake 发展，其组件逐渐被拆分为独立项目（如 [[trino]]、[[spark]]、[[apache-iceberg]]）。
