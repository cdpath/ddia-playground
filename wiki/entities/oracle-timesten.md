# Oracle TimesTen

**Oracle TimesTen** 是 Oracle 推出的 [[in-memory-database]]，主要面向需要极低延迟的金融、电信等 OLTP 场景。

## 特点

- 数据常驻内存，支持 SQL 和 ACID。
- 可作为独立数据库，也可作为 Oracle 数据库的缓存层。
- 通过 log 和 checkpoint 实现持久化。
