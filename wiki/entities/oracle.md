# Oracle Database

**Oracle Database** 是 Oracle 公司开发的商业关系型数据库管理系统（RDBMS）。

## 本章角色

- Oracle 也提供 [[in-memory-database]] 产品 [[oracle-timesten|Oracle TimesTen]]。
- 属于企业级 OLTP 和 HTAP 数据库供应商。
