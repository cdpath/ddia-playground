# BigQuery

**Google BigQuery** 是 Google Cloud 提供的 serverless [[data-warehouse]]。它将 storage 与 compute 分离，数据存放在对象存储中，用户按查询计算量付费。

## 本章角色

- 云原生 data warehouse 的代表。
- 内部采用 [[column-oriented-storage|列式存储]]。
- 与 Amazon Redshift、Snowflake 并列为主流云数据仓库。
