# SQL

**SQL（Structured Query Language）** 是最常用的关系型数据库 declarative query language，基于 [[relational-model]]。

## 特点

- 用 `SELECT`、`INSERT`、`UPDATE`、`DELETE` 操作数据。
- 支持 [[join]]、filter、aggregate、group、window functions、recursive CTE 等。
- 现代 SQL 已扩展支持 JSON、XML、graph 等数据类型。
