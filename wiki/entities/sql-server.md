# SQL Server

**Microsoft SQL Server** 是微软开发的关系型数据库管理系统。它既支持 [[online-transaction-processing|OLTP]] 工作负载，也支持数据仓库和分析工作负载，属于 [[hybrid-transactional-analytical-processing|HTAP]] 系统之一。

## 本章角色

- 支持显式指定一个 [[clustered-index]]。
- 提供 column store index 支持分析查询。
- 与 SAP HANA、SingleStore 并列为 HTAP 数据库。
