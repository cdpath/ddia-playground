# Apache Pinot

**Apache Pinot** 是面向实时分析的分布式 OLAP 数据库，常用于产品分析和监控场景。它采用 [[column-oriented-storage|列式存储]]。

## 本章角色

- 与 Druid、ClickHouse 并列为实时分析系统。
- 通过列式存储和索引加速高并发低延迟的聚合查询。
- 支持从 Kafka 等流式源实时摄取数据。
