# Presto

**Presto**（PrestoDB）是 Facebook（现 Meta）开发的开源分布式 SQL query engine，用于跨多种数据源执行交互式分析查询。

## 本章角色

- 现代 data lake query engine 的代表之一。
- 与 [[trino]] 同源但由不同社区维护。
