# ClickHouse

ClickHouse 是开源的列式 DBMS，针对 analytical query 优化。本书既把它列为 self-hosted analytical/OLAP 系统，也列为 real-time analytics 系统。

## 本章角色

- self-hosted OLAP engine 的例子
- 与 Pinot、Druid 并列为 product analytics / real-time analytics 系统
